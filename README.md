# BLUEBLUE
청와대 청원 게시판 분석 서비스 **(청청)**

## Description
- Crawler
    -  Selenium, BeautifulSoup4를 이용한 청와대 국민청원 게시판 Scraping

- Client
    - React.js를 이용한 국민청원 분석 결과 Visualization

- API
    - Flask를 이용한 국민청원 분석 결과 RestAPI

## Environment
- React.js
- Node.js
- npm
- Flask
- Beautifulsoup
- Selenium
